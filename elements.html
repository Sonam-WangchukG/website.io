Astro Lab 108

I chose the astro lab because I was curious about the universe and wanted to use my data science skills (Python, NumPy, Matplotlib) on real celestial data. My goal was to learn the methodologies astronomers use, blending programming with observational science. The course successfully diversified my knowledge by combining computer science and astrophysics. I improved my Python skills for data processing and gained practical expertise in key methods like transit detection (for planet size) and the radial velocity method (for planet mass), using data from satellites like TESS. This taught me Astroinformatics, making complex research feel achievable.
