Week 1

During the first week of astro lab, I learned that the foundational tools for handling and visualizing astronomical data are Python, NumPy, and Matplotlib. Python is the core programming language I use; NumPy simplifies all the math and data handling by using fast array structures; and Matplotlib provides the ability to draw various graphs, such as \ line and scatter plots, so I can effectively see and analyze the data.

Week 2

During the second week, I learned the two main methods for characterizing exoplanets. First, the Transit Method uses the drop in a star's brightness, recorded in a light curve, to find the transit depth, which then tells me the planet's size (radius). Second, the Radial Velocity Method measures the star's slight wobble using the Doppler shift of its light; this wobble is used to calculate the planet's mass, which I can express in Jupiter or Earth masses. By combining these two techniques, I gain a complete picture of the exoplanet, including its density

Week 3

During the third week of the lab, I applied my skills to real astronomical data to search for exoplanets detected by NASA's TESS satellite using the Transit Method. I learned that TESS finds planets by observing small, rhythmic drops in starlight. Using the lightkurve Python library, I practiced the core steps of analysis: I downloaded and cleaned the star's light curve, found the planet's orbital period, and folded the data to stack all the transit events, which allowed me to confirm and clearly visualize the faint planet signal.

Week 4

During the fourth week, I learned Git, a powerful tool for version control that tracks changes in code and data. I learned that platforms like GitHub or GitLab host projects in repositories. I practiced core Git concepts: saving snapshots of work using commits , creating branches to work on features without affecting the main codebase , and merging changes. I also learned key commands like git clone (to download a project) , git commit (to save local changes) , and git push (to upload my work). This established a professional workflow for collaborative coding..

Week 5

During the final week, I learned essential tools for both collaboration and presentation. I mastered Git and GitHub for version control, using commands like git commit and git push to manage project code. I learned LaTeX and used Overleaf to create standard, professional-looking CVs and resumes, which is vital for academic documentation.
